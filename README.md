# VoluntraQ 🌍

**Empowering Communities, Coordinating Relief.**

VoluntraQ is a powerful, AI-driven crisis coordination platform designed to bridge the gap between NGOs and volunteers during disaster response. By leveraging real-time data and intelligent task matching, VoluntraQ ensures that the right help reaches the right place at the right time.

---

## 🚀 Features

- **🤖 AI Task Matching**: Intelligent volunteer-to-task assignment using Google Gemini 1.5 Pro.
- **📊 Real-time Dashboard**: Live coordination for NGO administrators and volunteers.
- **📍 Location-aware Services**: Integrated Google Maps for precise incident reporting and logistics.
- **🔐 Secure Auth**: Robust authentication system with Firebase integration.
- **📱 Cross-Platform**: Built with Flutter for a seamless experience on Android and iOS.

---

## 🛠 Tech Stack

- **Frontend**: Flutter / Dart
- **Backend**: Node.js / Express
- **AI**: Google Gemini API
- **Database**: Firebase Realtime Database
- **Auth**: Firebase Authentication

---

## 📦 Getting Started

### Backend Setup
1. Navigate to `voluntraq_backend`
2. Run `npm install`
3. Create a `.env` file with your `GEMINI_API_KEY` and Firebase credentials.
4. Run `npm run dev`

### Frontend Setup
1. Navigate to `voluntraq_frontend`
2. Run `flutter pub get`
3. Run `flutter run`
